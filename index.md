Политика конфиденциальности приложения "В ожидании чуда: мальчик или девочка"

Общие положения. Настоящая политика обработки персональных данных составлена в соответствии с требованиями ФЗ от 27.07.2006 №152-ФЗ «О персональных данных». Приложение «В ожидании чуда» (далее — Приложение) не собирает и не передает персональные данные пользователей третьим лицам.
Данные. Все вводимые пользователем данные (даты рождения, даты зачатия) обрабатываются исключительно на устройстве пользователя и не сохраняются на внешних серверах.
Аналитика. Приложение может использовать анонимные данные (модель телефона, версия ОС) для улучшения работы через сервисы:
 - AppMetrica (ООО «ЯНДЕКС»): для анализа ошибок и использования приложения. https://yandex.ru/legal/confidential/ru/
 - MyTracker (ООО «ВК»): для анализа ошибок и использования приложения. https://tracker.my.com/ru/legal/privacy-policy
 - MyTarget / VK Реклама (ООО «ВК»): для показа персонализированной рекламы https://target.vk.ru/help/advertisers/privacy_policy/
Файлы Cookie и рекламные ID. Сторонние сервисы могут использовать рекламный идентификатор (GAID) вашего устройства для предоставления релевантной рекламы и анализа трафика.
Безопасность
Мы ценим ваше доверие, поэтому используем общепринятые средства защиты данных. Помните, что ни один метод передачи через интернет не является на 100% безопасным.
